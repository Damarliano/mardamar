Setup Supabase

Ikuti langkah berikut agar foto & video tersimpan online — bisa dilihat dari perangkat apapun tanpa login.

1
Buat akun gratis di supabase.com -> klik New Project.
2
Tunggu project siap -> buka Project Settings -> API -> copy Project URL dan anon public key.
3
Buka SQL Editor -> salin perintah SQL dari komentar di atas file HTML ini -> klik Run.
4
Edit file HTML ini -> cari CONFIG di baris paling atas -> isi SUPABASE_URL dan SUPABASE_ANON -> simpan & upload ulang ke GitHub.
✅ Guest (siapapun) bisa melihat semua foto & video tanpa login, dari perangkat apapun.
🔐 Owner (Damar) login dengan password untuk upload & hapus konten.
