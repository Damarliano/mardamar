Import dari Google Drive

Untuk terhubung ke Google Drive, masukkan API Key dan Client ID dari Google Cloud Console. Setup ini hanya perlu dilakukan sekali — tersimpan otomatis di browser.

1
Buka Google Cloud Console, buat project baru atau pilih yang sudah ada.
2
Aktifkan Google Drive API dan Google Picker API di menu APIs & Services → Library.
3
Buat API Key di Credentials, lalu buat OAuth 2.0 Client ID (tipe: Web Application). Tambahkan URL website ini ke Authorized JavaScript Origins.
4
Salin kedua nilai di bawah lalu klik Hubungkan. Setelah terhubung, kamu bisa pilih foto & video langsung dari Drive.
